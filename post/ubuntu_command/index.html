Ubuntu Base Command ( 普段よく使われているCommand )

収録中ing »

ubuntu update and upgrade

sudo apt-get update & sudo apt-get upgrade

ubuntu install package

sudo apt-get install *

ubuntu folder create

sudo mkdir <dir name>
# ubuntu delete dir 
sudo rmdir <dir name>
# ubuntu delete dir and file
sudo rm -r <dir name>

ubuntu ufw firewall

sudo apt-get install ufw
sudo ufw enable
# sudo ufw allow 8080 <-例
sudo ufw allow <port>
sudo ufw deny <port>
sudo ufw status
sudo ufw disable

ubuntu file manager gui

# ubuntu lfm
sudo apt-get install lfm

Ubuntu file download and upload

sudo scp [file path] [username]@[ip]:[folder path]

Ubuntu Run Python use nohup

nohup python -u [py name] > [log name].log 2>&1 &

GNU计划开发的用于命令行终端切换的自由软件。

sudo apt install screen
# create screen
screen -S <name>
# screen list
screen -ls
# screen resume
screen -r <name>

Ubuntu DiskUtil

# ubuntu check disk information
df -hl
# check all hardware
lsblk
# lshw is check hardware information
sudo lshw -class disk

Ubuntu Wget

sudo apt install wget
